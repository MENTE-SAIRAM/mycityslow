# My City Slow — Logo Design Brief

## Project Overview
**App Name:** My City Slow  
**Tagline:** Discover the Peaceful Side of Your City  
**Category:** Travel & Lifestyle Mobile App  
**Target Platform:** Android/iOS App Store, Website, Social Media

---

## Brand Identity

### Brand Essence
- **Core Value:** Slow living, peaceful exploration, hidden gems in Indian cities
- **Target Audience:** Mindful travelers, photographers, wellness seekers, cultural explorers (20-45 years, urban professionals)
- **Tone:** Serene, premium, trustworthy, modern, culturally conscious

### Visual Philosophy
- Minimal & elegant
- Nature-inspired with subtle urban elements
- Calming and harmonious
- Modern Indian touch (not stereotypical)
- High attention to detail & negative space
- Premium, app-store ready quality

## Core Design Elements

### Visual Metaphors to Incorporate
1. **Window Frame** — represents "hidden peaceful spots" (discovering hidden places)
2. **Tree / Leaf** — nature, calm, peace
3. **Lotus Flower** — spirituality, slow living, peace
4. **Gentle Hill** — landscape, journey, peaceful mountains
5. **Subtle City Skyline** — modern Indian cities
6. **Bird in Flight** — freedom, exploration

### Design Principles
- **Balanced:** Symmetrical or well-distributed composition
- **Negative Space:** Clean, breathing space around elements
- **Layered:** Subtle depth without complexity
- **Scalable:** Must work at 16px (app icon) to 2000px (print)
- **Modern:** Contemporary sans-serif, no dated elements
- **Cultural Nod:** Subtle Indian influence (geometry, motifs) without clichés

---

## Logo Variations Required

### 1. **Main Logo with Text** (Horizontal)
**Use:** Website header, social media banner, email footer, print collateral

**Specifications:**
- Dimensions: 1200px W x 400px H (or 3:1 ratio)
- Format: PNG (transparent), SVG (vector), PDF
- Icon on left, text on right (or icon above text)
- Icon size: ~200px (icon portion)
- Text: "My City Slow" in elegant sans-serif (medium weight)
- Spacing: Adequate white space (min 30px padding all sides)
- Version: Full color (RGB) + Black & White (B&W for printing)

**Design Notes:**
- Text should use Deep Forest (#3D5A3D) or Black
- Icon uses primary colors (Sage green, terracotta, beige)
- Tagline optional below main text: "Discover the Peaceful Side of Your City" in smaller, lighter weight

---

### 2. **Icon-Only Version** (App Icon / Square)
**Use:** Android app icon, iOS app icon, favicon, profile picture, app launcher

**Specifications:**
- Dimensions: 512px x 512px (square, 1:1 ratio)
- Format: PNG (transparent), SVG
- Safe Area: Main icon content within 400px x 400px (leaving 56px padding for iOS/Android safety margins)
- Rounded Corners: 110px corner radius (for iOS/iPadOS)
- Background: Optional solid background (suggest light cream #F5F1ED or transparent)
- No text on icon version
- Versions needed:
  - Square (512x512) — app stores
  - Round (512x512 with circular mask) — app launcher
  - Transparent background
  - Solid background option

**Design Notes:**
- Must be recognizable at 32px (smallest size on app launcher)
- Must be recognizable at 180px (App Store)
- Should feel premium, not generic
- Avoid thin strokes (min 2px at 512px size)
- High contrast for readability

---

### 3. **Horizontal Version** (Website / Desktop)
**Use:** Website header logo, landing page, marketing materials, presentations

**Specifications:**
- Dimensions: 1600px W x 600px H (or similar wide format, 2.67:1 ratio)
- Format: PNG, SVG, PDF
- Icon + Horizontal Text Layout
- Icon size: ~300px
- Spacing: Clean white space around composition
- Text: "My City Slow" with optional tagline below
- Versions: Full color + Black & White + Single color (for flexible use)

**Design Notes:**
- Website version can include subtle animation capability (SVG hints)
- Should scale well from 200px width to 1600px width
- Professional enough for app store screenshots and marketing website

---

## Additional Design Requirements

### Typography
- **Font Family:** Modern, elegant sans-serif
  - Recommended: Inter, Poppins, Outfit, Space Grotesk (or similar)
  - Weight: Medium (500) for main text, Light (300) for tagline
  - NOT Comic Sans, Gotham (too corporate), or script fonts
- **Spacing:** Generous letter-spacing for elegance
- **Case:** Title case ("My City Slow")

### Icon/Illustration Style
- **Stroke Width:** Consistent, ~2-3px at standard size
- **Style:** Minimal line art or solid geometric shapes
- **Illustration Type:** Not photorealistic, not cartoon, not flat emoji-style
- **Line Quality:** Smooth curves, professional vector quality

### Flexibility Requirements
- Must work well with both app colors AND in grayscale
- Must work on light backgrounds AND dark mode
- Must remain legible when scaled down to 24px
- Must look good on both iOS and Android UI

## Design Process & Feedback

### Recommended Approach
1. **Concept Phase:** 3-5 unique initial concepts exploring window, tree, lotus, and city metaphors
2. **Selection:** Client selects preferred direction(s)
3. **Refinement:** Polish chosen concept, refine colors, test at various sizes
4. **Variations:** Create all required versions (icon, horizontal, website)
5. **Testing:** Verify on app mockup, website mockup, small sizes

### Feedback Rounds
- Concept presentation (initial ideas)
- First round refinement (2 rounds)
- Final revision (unlimited until approval)

## Usage Guidelines (For Developer Implementation)

### Android Implementation
- **App Icon:** 512x512 PNG (transparent) → Android Studio will auto-generate all sizes
- **Drawable Placement:** `app/src/main/res/drawable/logo_main.xml` (or PNG)
- **Onboarding Screens:** Display at 120dp, 100dp sizes with rounded corners
- **Home Screen:** Display at 80dp with subtle shadow

### iOS Implementation
- **App Icon:** 180x180 PNG (both with transparency and solid background options)
- **LaunchScreen:** Display logo at 150x150 size
- **Settings/About:** Display at 100x100 size

### Web Implementation
- **Header:** Horizontal version at 300px width (responsive)
- **Social Media:** 1200x630px with logo + text for sharing
- **Favicon:** SVG + PNG versions
